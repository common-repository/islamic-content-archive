== Description ==

Islamic Content Archive is a plugin that allows you to get the content (articles, videos, audios) of 27 Islamic websites in different languages using Json API. The data is stored directly in the database of your site according to the category and the language.
All what you have to do is to install the Plugin, select the language, choose the site, then the content will be stored automatically in your database under the category you choose or according to the original category. Afterwards, the plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.

Features:

* Get the content (articles, videos, audios) of 27 Islamic websites in different languages
* More than 25 Islamic websites
* Using Json API
* Using Cron Job.


PS: All the websites owned by EDC [(E-Da`wah Committee)](https://edc.org.kw), For Support click to [Islam.com.kw](https://islam.com.kw/).


== Installation ==

* Upload the Plugin to your blog, Activate it.
* Select the language.
* Choose the site.
* Then the content will be stored automatically in your database under the category you choose or according to the original category.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
